# Uniform Unit Allowance Management System

This system is developed using Java (JSP & Servlet), MySQL, and Apache Tomcat.

## Features
- Track attendance of unit members
- Calculate allowance based on attendance hours
- Manage payment status

## Technologies Used
- Java (JSP & Servlet)
- Apache Tomcat
- MySQL (XAMPP)

## How to Run
1. Import project into NetBeans
2. Start XAMPP (Apache & MySQL)
3. Import database.sql into phpMyAdmin
4. Run using Apache Tomcat
