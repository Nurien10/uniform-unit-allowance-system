# UMT Uniform Units Allowance Management System

The UMT Uniform Units Allowance Management System is a web-based platform designed to streamline attendance tracking and allowance distribution for uniformed units at Universiti Malaysia Terengganu (UMT). The system replaces the manual WhatsApp and Excel-based process with a structured digital solution to improve accuracy, transparency, and efficiency.

---

# 📌 1. Project Background

The system was developed to address administrative inefficiencies in managing attendance and allowance distribution for uniform units at UMT.

Previously, attendance and allowance tracking were done manually through informal communication (WhatsApp) and Excel sheets, leading to errors, missing records, and inconsistent allowance calculations.

This system provides a centralized platform where:
- Unit High Council manages program schedules
- Unit Members can view programs and confirm attendance
- Unit Trainers verify attendance and approve records
- External Officers handle allowance payments

By digitizing the process, the system improves accuracy, reduces manual workload, and ensures fair allowance distribution based on verified participation.

---

# 📌 2. Problem Statement

The current manual system leads to:
- Inconsistent attendance records
- Human errors in data entry
- Difficulty tracking participation
- Incorrect allowance calculations
- Lack of transparency in financial distribution

These issues affect fairness, efficiency, and trust among stakeholders.

---

# 📌 3. Proposed Solution

The proposed system introduces a structured workflow:

- UHC creates and manages program schedules
- UM views programs, confirms attendance, and uploads proof if absent
- UT verifies and approves attendance records
- System automatically calculates total attendance hours
- Allowance is calculated based on verified attendance
- EO processes final payment distribution

This ensures transparency, accuracy, and accountability in the entire process.

---

# 📌 4. Objectives

- Identify stakeholder requirements (UM, UHC, UT, EO)
- Design a web-based attendance and allowance system
- Develop a centralized platform for attendance tracking and allowance calculation
- Improve efficiency, transparency, and data accuracy
